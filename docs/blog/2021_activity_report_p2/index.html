2021е№ҙеәҰгҒ®жҙ»еӢ•е ұе‘Ҡpart2

2022.04.11

гҒ“гӮ“гҒ«гҒЎгҒҜпјҒ

CORE15thд»ЈиЎЁгҒ®гҒ‘гӮ“TгҒ§гҒҷгҖӮ

еүҚеӣһгҒ«еј•гҒҚз¶ҡгҒҚгҖҒ2021е№ҙеәҰгҒ®жҙ»еӢ•гӮ’жҢҜгӮҠиҝ”гӮҠгҒҫгҒҷгҖӮ

з¬¬20еӣһдјҠиұҶеӨ§еі¶е…ұеҗҢжү“дёҠе®ҹйЁ“

гҖҖиғҪд»ЈгҒ§гҒҜгғҒгғЈгғ¬гғігӮёгғігӮ°гҒӘгҒ“гҒЁгӮ’зӣ®жҢҮгҒ—гҒҫгҒ—гҒҹгҒҢгҖҒдјҠиұҶеӨ§еі¶гҒ§гҒҜгҒҶгҒЈгҒҰгҒӢгӮҸгҒЈгҒҰгӮ·гғігғ—гғ«гҒ§зўәе®ҹгҒ«жҲҗеҠҹгҒҷгӮӢгғӯгӮұгғғгғҲгӮ’дҪңгӮҚгҒҶгҒЁй ‘ејөгӮҠгҒҫгҒ—гҒҹгҖӮиғҪд»ЈгҒ§гҒ®еӨұж•—гӮ’з”ҹгҒӢгҒ—гҒӘгҒҢгӮүгҖҒ2021е№ҙеәҰгҒ«ж–°гҒ—гҒҸе…ҘдјҡгҒ—гҒҹгғЎгғігғҗгғјгҒёгҒ®жҠҖиЎ“з¶ҷжүҝгӮ’зӣ®зҡ„гҒ«гҖҢж–°е…Ҙз”ҹгғ—гғӯгӮёгӮ§гӮҜгғҲгҖҚгӮ’з«ӢдёҠгҒ’гҒҫгҒ—гҒҹгҖӮ

гҖҖгҒқгҒ®гғҹгғғгӮ·гғ§гғігҒҜжң¬еҪ“гҒ«гҒ„гҒҹгҒЈгҒҰгӮ·гғігғ—гғ«гҒ§гҖҒгғӯгӮұгғғгғҲгҒ®дёҠжҳҮгҒЁдёӢйҷҚгӮ’и©•дҫЎгҒҷгӮӢгҖҒгғӯгӮұгғғгғҲгҒ«жҗӯијүгҒ—гҒҹйӣ»иЈ…жң¬дҪ“еӣһеҸҺгҒЁгғҮгғјгӮҝгҒ®еӣһеҸҺгӮ’и©•дҫЎгҒҷгӮӢгҒЁгҒ„гҒҶгғҹгғғгӮ·гғ§гғіеҶ…е®№гҒ§гҒ—гҒҹгҖӮгҒҫгҒҹCOREгҒҜгҒ“гҒ“ж•°е№ҙгҖҒзӢ¬иҮӘй–ӢзҷәгӮЁгғігӮёгғігӮ’гғӯгӮұгғғгғҲгҒ«жҗӯијүгҒҷгӮӢгҒ“гҒЁгҒҢеӨҡгҒӢгҒЈгҒҹгҒ§гҒҷгҒҢгҖҒеҹәжң¬гҒ«з«ӢгҒЎиҝ”гӮӢгҒЁгҒ„гҒҶж„Ҹе‘ігҒ§Hyper TEKгҒЁе‘јгҒ°гӮҢгӮӢгӮЁгғігӮёгғігӮ’жҺЎз”ЁгҒ—гҒҫгҒ—гҒҹгҖӮгҒ—гҒӢгҒ—гҖҒгҒқгӮҢгҒ гҒ‘гҒ§гҒҜгҒӘгҒҸ360еәҰгӮ«гғЎгғ©гӮ’гғҺгғјгӮәе…Ҳз«ҜгҒ«жҗӯијүгҒ—гҒҰж’®еҪұгӮ’и©ҰгҒҝгӮӢгҒЁгҒ„гҒЈгҒҹCOREгӮүгҒ—гҒ„гӮөгғ–гғҹгғғгӮ·гғ§гғігӮӮгҒӮгӮҠгҒҫгҒ—гҒҹгҖӮ

гҖҖж©ҹдҪ“еҗҚгҒҜгҖҢCANVASгҖҚгҒ§гҒҷгҖӮеҗҚеүҚгҒ®з”ұжқҘгҒҜгғӯгӮұгғғгғҲйӣ»иЈ…гҒ«йҖҡдҝЎиҰҸж јгҒ®гҒІгҒЁгҒӨгҒ§гҒӮгӮӢCANгҒЁгҒ„гҒҶгӮӮгҒ®гҒ§гҒҷгҖӮCANйҖҡдҝЎгҒҜиҮӘеӢ•и»ҠгҒ®гӮўгғ“гӮӘгғӢгӮҜгӮ№гҒӘгҒ©гҒ«жҺЎз”ЁгҒ•гӮҢгҒҰгҒ„гҒҰгҖҒеҫ“жқҘгҒ®йҖҡдҝЎгҒ®гӮҲгҒҶгҒӘиҰӘж©ҹгғ»еӯҗж©ҹгҒ®й–ўдҝӮгҒ§гҒҜгҒӘгҒҸгғһгғ«гғҒгғһгӮ№гӮҝгғјгҒЁе‘јгҒ°гӮҢгӮӢе…ЁйғЁиҰӘж©ҹеҗҢеЈ«гҒ®йҖҡдҝЎгҒЁгҒ„гҒЈгҒҹгҒ“гҒЁгҒҢеҸҜиғҪгҒ§гҒҷгҖӮгҒҫгҒҹеӨ–йғЁгҒӢгӮүгҒ®гғҺгӮӨгӮәгҒ«еј·гҒ„гҒЁгҒ„гҒҶзү№еҫҙгӮ„й…Қз·ҡгӮ’е°‘гҒӘгҒҸгҒ§гҒҚгӮӢгҒЁгҒ„гҒҶеҲ©зӮ№гҒҢгҒӮгӮҠгҒҫгҒҷгҖӮгҒ“гҒ®гғӯгӮұгғғгғҲгҒ®гғҗгӮ№(йӣ»иЈ…йғЁ)гҒҜCANгӮ’дҪҝгҒЈгҒҰгӮӢгҒЁгҒ„гҒҶгҒ“гҒЁгҒ§гҖҢCANVASгҖҚгҒ§гҒҷгҖӮ(гғӯгӮұгғғгғҲгӮ’еЎ—иЈ…гҒ—гҒҹгҒЁгҒҚжңҖеҲқиӘӨгҒЈгҒҰгҖҢCANBUSгҖҚгҒЁз¶ҙгҒЈгҒҰгҒ—гҒҫгҒЈгҒҹгҒ®гҒ§гҒҷгҒҢгҖҒгғҗгӮ№гҒ®иӢұиӘһгҒҜbusгҒӘгҒ®гҒ§гҒӮгҒӘгҒҢгҒЎй–“йҒ•гҒЈгҒҰгҒӘгҒӢгҒЈгҒҹгҒ®гҒ§гҒҜпјҹгҒЁгҒ„гҒҶи©ұгҒ§зӣӣгӮҠдёҠгҒҢгӮҠгҒҫгҒ—гҒҹз¬‘)

гҖҖгҒ•гҒҰгҖҒе®ҹйҡӣгҒ®жү“дёҠгҒ’гҒ§гҒҷгҒҢд»ҠеӣһгӮӮгғҸгғ—гғӢгғігӮ°гҒӢгӮүе§ӢгҒҫгӮҠгҒҫгҒ—гҒҹгҖӮ

гҖҖгҒқгҒҶгҒ§гҒҷгҖӮгҒҫгҒҹзҸҫең°еҜ©жҹ»гҒ§гҒҷгҖӮеүҚеӣһгҒ®иғҪд»Је®Үе®ҷгӮӨгғҷгғігғҲгҒ§гҒҜйҮҚйҮҸгӮӘгғјгғҗгғјгҒ§дёҚеҗҲж јгӮ’дёҖеәҰгӮӮгӮүгҒҶгҒЁгҒ„гҒҶеӨұж…ӢгӮ’зҠҜгҒ—гҒҹгҒҹгӮҒгҖҒгғӯгӮұгғғгғҲгҒ®йҮҚйҮҸиЁҲжё¬гҒЁйҮҚеҝғдҪҚзҪ®гҒ®зҸҫең°е…ҘгӮҠеүҚгҒ®жё¬е®ҡгҒ«гҒҜеҝөгҒ«гҒҜеҝөгӮ’е…ҘгӮҢгҒҫгҒ—гҒҹгҖӮгҒқгҒ®гҒҹгӮҒд»ҠеӣһгҒҜйҮҚйҮҸгҒ®ж–№гҒҜе•ҸйЎҢгҒӘгҒҸгӮҜгғӘгӮўгҒ§гҒҚгҖҒгғӣгғғгғҲгҒ—гҒҹгҒ®гӮӮгҒӨгҒӢгҒ®й–“гҖҒд»ҠеәҰгҒҜгғ•гӮЈгғігҒ®еҜёжі•гҒҢе®ҢжҲҗе ұе‘ҠжӣёгҒ®гӮӮгҒ®гҒЁйҒ•гҒҶгҒ“гҒЁгҒҢзҷәиҰҡгҒ—гҒҫгҒ—гҒҹгҖӮгғ•гӮЈгғігҒҜгғӯгӮұгғғгғҲгҒ®йЈӣзҝ”гҒ«й–ўгӮҸгӮӢеӨ§дәӢгҒӘгғ‘гғјгғ„гҒ§гҒӮгӮӢгҒҹгӮҒгҖҒеҜёжі•йҒ•гҒ„гҒҜеӨ§гҒҚгҒӘдәӢд»¶гҒ§гҒҷгҖӮеҺҹеӣ гҒҜгғ•гӮЈгғігҒ®еҪўзҠ¶гӮ’иЁӯиЁҲгҒҷгӮӢгӮ·гғҹгғҘгғ¬гғјгӮ·гғ§гғіжӢ…еҪ“гҒЁе®ҹйҡӣгҒ«гғ•гӮЈгғігӮ’дҪңгҒЈгҒҹж§ӢйҖ зҸӯе“ЎгҒ®иӘҚиӯҳгҒ®гҒҡгӮҢгҒ§гҒ—гҒҹгҖӮиЈҪдҪңж®өйҡҺгҒ§гӮ·гғҹгғҘгғ¬гғјгӮ·гғ§гғіжӢ…еҪ“гҒӢгӮүгғ•гӮЈгғігҒ®еҪўзҠ¶еӨүжӣҙгҒ®жҢҮзӨәгҒҢгҒӮгҒЈгҒҹгҒҢгҖҒгҒқгҒ®еӨүжӣҙеҶ…е®№гҒҢжӯЈгҒ—гҒҸиӘҚиӯҳгҒ•гӮҢгҒҡиӘӨгҒЈгҒҹеҜёжі•гҒ®гғ•гӮЈгғігӮ’дҪңгҒЈгҒҰгҒ—гҒҫгҒЈгҒҹгҒЁгҒ„гҒҶж„ҹгҒҳгҒ§гҒҷгҖӮгҒ“гҒ®гҒ“гҒЁгҒӢгӮүе®ҢжҲҗе ұе‘ҠжӣёгӮ’жҸҗеҮәгҒҷгӮӢеүҚгҒ«иЎҢгҒҶж§ҳгҖ…гҒӘиЁҲжё¬гҒ«гҒҠгҒ„гҒҰгӮ„гӮҠж–№гӮ’ж”№гӮҒгҒӘгҒ„гҒЁгҖҒдёҖгҒӨе•ҸйЎҢгӮ’гҒӨгҒ¶гҒ—гҒҰгӮӮгҒҫгҒҹж–°гҒҹгҒӘе•ҸйЎҢгҒҢзҷәз”ҹгҒҷгӮӢгҒЁгҒ„гҒҶгҒ“гҒЁгҒҢеҲҶгҒӢгӮҠгҒҫгҒ—гҒҹгҖӮд»ҠеӣһгӮӮе®ҹйЁ“йҒӢе–¶гҒ®ж–№гҖ…гӮ„еҜ©жҹ»е“ЎгҒ®ж–№гҖ…гҒ®гҒ”еҺҡж„ҸгҒ§гҖҒзҸҫең°гҒ§е®ҹжё¬гҒ—гҒҹгғ•гӮЈгғігҒ®еҜёжі•гҒ§гӮӮгҒҶдёҖеәҰCFD(ж•°еҖӨжөҒдҪ“еҠӣеӯҰгҒ®гӮ·гғҹгғҘгғ¬гғјгӮҝ)гӮ’гҒҫгӮҸгҒ—гҒҰгғӯгӮұгғғгғҲгҒ®йўЁең§дёӯеҝғгӮ’еҮәгҒ—гҒӘгҒҠгҒҷгҒЁгҒ„гҒҶеҜҫеҮҰгҒ§иЁұгҒ—гҒҰгҒ„гҒҹгҒ гҒ‘гҒҫгҒ—гҒҹгҖӮжң¬еҪ“гҒ«ж„ҹи¬қгҒ—гҒҰгҒ„гҒҫгҒҷгҖӮ

гҖҖгҒқгҒ—гҒҰжқҘгӮӢжү“дёҠгҒ’еҪ“ж—ҘгҒ§гҒҷгҒҢгҖҒеӨ©еҖҷ(зү№гҒ«йўЁ)гҒҢжңҹй–“дёӯеҝғй…ҚгҒ•гӮҢгҒҰгҒ„гӮӢдёӯгҖҒе…Ё6ж©ҹдҪ“гҒ®жү“дёҠгҒ’гӮ’гҒҷгӮӢгҒ«гҒҜжү“гҒҰгӮӢгғҒгғЈгғігӮ№гҒ§еҝ…гҒҡжү“гҒӨпјҒгҒЁгҒ„гҒҶгҒ®гҒҢйҮҚиҰҒгҒ«гҒӘгҒЈгҒҰгҒҫгҒ—гҒҹгҖӮгҒ—гҒӢгҒ—гҖҒж–°е…Ҙз”ҹгҒҢдёӯеҝғгҒЁгҒӘгҒЈгҒҹгғЎгғігғҗгғјгҒҢдёҠзҙҡз”ҹгҒ®еҠ©гҒ‘гӮӮеҸ—гҒ‘гҒӘгҒҢгӮүжә–еӮҷгӮ’зқҖгҖ…гҒЁйҖІгӮҒгҖҒзӮ№зҒ«гӮ·гғјгӮұгғігӮ№гҒҫгҒ§жҢҒгҒЈгҒҰгҒҸгӮӢгҒ“гҒЁгҒҢеҮәжқҘгҒҫгҒ—гҒҹгҖӮ

гҖҖгҒқгҒ—гҒҰйҒӢе‘ҪгҒ®зһ¬й–“гҖӮ2022е№ҙ3жңҲ19ж—Ҙ14жҷӮ30еҲҶгҖҒгғҸгӮӨгғ–гғӘгғғгғүгғӯгӮұгғғгғҲгҖҢCANVASгҖҚгҒҜдјҠиұҶеӨ§еі¶иЈҸз Ӯжј гҒ§иҰӢдәӢжү“дёҠгҒ’гҒ«жҲҗеҠҹгҒ—гҒҫгҒ—гҒҹпјҒCOREгҒЁгҒ—гҒҰгҒҜзҙ„1е№ҙгҒ¶гӮҠгҖҒдјҠиұҶеӨ§еі¶гҒ«гҒҠгҒ„гҒҰгҒҜCOREгҒҜ2е№ҙеҚҠгҒ¶гӮҠгҒ«жү“дёҠгҒ’гҒ«жҲҗеҠҹгҒ—гҒҫгҒ—гҒҹгҖӮгҒҫгҒҹд»ҠеӣһгҒҜгҒқгӮҢгҒ гҒ‘гҒ§гҒҜгҒӮгӮҠгҒҫгҒӣгӮ“пјҒй–Ӣж”ҫж©ҹж§ӢгҒҢдҪңеӢ•гҒ—гҖҒз„ЎдәӢгғ‘гғ©гӮ·гғҘгғјгғҲгӮ’ж”ҫеҮәгҒ—гҒҰжёӣйҖҹиҗҪдёӢгҒ•гҒӣгӮӢгҒ“гҒЁгӮӮжҲҗеҠҹгҒ—гҒҫгҒ—гҒҹгҖӮгғ‘гғ©гӮ·гғҘгғјгғҲй–ӢеӮҳгҒҜгҒ“гҒ“ж•°е№ҙCOREгҒ®иӘІйЎҢгҒ§гҒ—гҒҹгҒҢгҖҒиҰӢдәӢжҲҗеҠҹгҒ—гҒҫгҒ—гҒҹгҖӮеғ•гӮӮCOREгҒ«е…ҘгҒЈгҒҰ3е№ҙзөҢгҒЎгҒҫгҒҷгҒҢгҖҒжңҖй«ҳгҒЁиЁҖгҒЈгҒҰгҒ„гҒ„жү“дёҠгҒ’гҒ«гҒӘгҒЈгҒҹгҒЁжҖқгҒ„гҒҫгҒҷгҖӮгӮӮгҒЎгӮҚгӮ“гғӯгӮұгғғгғҲгҒҜз„ЎдәӢеӣһеҸҺгӮӮгҒ•гӮҢгҖҒгҒҹгҒҸгҒ•гӮ“гҒ®гғҮгғјгӮҝгӮ’ж®ӢгҒ—гҒҰгҒҸгӮҢгҒҫгҒ—гҒҹгҖӮзү№гҒ«зӣ®зҺүгҒ§гҒӮгҒЈгҒҹ360еәҰгӮ«гғЎгғ©гҒ®жҳ еғҸгҒҜең§е·»гҒ§гҒҷгҖӮ
